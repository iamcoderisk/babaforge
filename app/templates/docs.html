API Documentation

Everything you need to integrate SendBaba into your applications

Quick Start

1. Get Your API Key

Sign up and navigate to your dashboard to find your API key. Include it in the X-API-Key header for all requests.

2. Base URL

https://sendbaba.com/api/v1

API Endpoints

POST /send

Send Single Email

Send a single email to one recipient

Request Body:

{
  "to": "recipient@example.com",
  "from": "sender@yourdomain.com",
  "subject": "Hello World",
  "html_body": "
Hello!
This is a test email.
",
  "text_body": "Hello! This is a test email."
}

Example (cURL):

curl -X POST https://sendbaba.com/api/v1/send \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "recipient@example.com",
    "from": "hello@yourdomain.com",
    "subject": "Test Email",
    "html_body": "
Hello World!
"
  }'
POST /send/bulk

Send Bulk Emails

Send multiple emails in one request (up to 10,000)

Request Body:

{
  "emails": [
    {
      "to": "user1@example.com",
      "from": "hello@yourdomain.com",
      "subject": "Welcome!",
      "html_body": "
Welcome!
"
    },
    {
      "to": "user2@example.com",
      "from": "hello@yourdomain.com",
      "subject": "Welcome!",
      "html_body": "
Welcome!
"
    }
  ]
}
POST /contact

Add Contact

Add a new contact to your list

Request Body:

{
  "email": "contact@example.com",
  "first_name": "John",
  "last_name": "Doe",
  "company": "Acme Inc",
  "tags": ["customer", "vip"]
}
GET /contacts

List Contacts

Get all your contacts with pagination

Query Parameters:

  • page - Page number (default: 1)
  • per_page - Items per page (default: 50, max: 100)
  • search - Search term
GET /usage

Get Usage Stats

Check your current email usage and limits

Response:

{
  "success": true,
  "usage": {
    "today": {
      "sent": 1250,
      "failed": 3
    },
    "last_30_days": {
      "sent": 45000,
      "failed": 120
    }
  }
}
